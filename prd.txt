# PRD.md
# Project: AI Interview Simulator
Version: 1.0
Status: MVP
Goal: Build an AI-powered interview preparation platform using MiroFish agents that analyzes a user's CV and a target job description, simulates multiple interviewers, and generates actionable feedback with 0-cost infrastructure.

---

# 1. Problem

Job seekers repeatedly apply without knowing:

- which questions they will face
- weaknesses inside their CV
- missing skills
- hiring risks
- how recruiters perceive them

Current interview tools mostly generate generic questions and lack realistic multi-perspective simulations.

---

# 2. Solution

Build a web platform where users upload:

- CV
- job description

MiroFish creates AI agents:

1. HR interviewer
2. Technical interviewer
3. Team manager
4. CEO
5. Critical interviewer

Agents independently analyze the candidate and interact together.

System outputs:

- expected interview questions
- CV weak points
- missing skills
- confidence score
- recruiter concerns
- improvement plan
- interview simulation
